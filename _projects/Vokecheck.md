# What is Vokecheck?
[Vokecheck](https://github.com/ChrompyCoder/vokecheck) - A CronJob scheduler tool.

# Check out the project [here](https://github.com/ChrompyCoder/vokecheck)

# Why this project?
To learn how automation works in the industry. This project uses fastAPI to send the jobs to celery worker through redis and stores all the contents in mongodb.

# Some features - 
- Create, schedule and run cronjobs.
- Extremely fast.
- GUI and CLI(through api).
